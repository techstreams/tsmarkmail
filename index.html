About

Overview


Want to create markdown formatted email and send it directly from a Google DriveTM Document?

Try TSMarkMail!



Features


  • Install and access TSMarkMail from the convenient Google Document Add-ons menu.

  • Author, send and reuse markdown content from any Google Document.

  • Easily select recipients with Google Contacts integration.

  • Automatically include images in email.

  • Quickly clear document content to begin a new email.

  • Work together with Google Document Chat and Real-Time Collaboration.

  • Create and update content from your mobile device.
